IMPORTANTE:
En el momento de registrar a un nuevo usuario desde el apartado ¡REGISTRATE! Del login de mi parte del Panel de Administración, 
ese usuario se registra de forma automática como un Usuario normal, ya que al comentárselo a Enrique me dijo que SIEMPRE un usuario 
debe registrase como un usuario normal para poder gestionar sus casas/componentes etc, y no obviamente como un Administrador.

Para poder acceder al panel de Administración con un usuario creado, es necesario entrar a PhPMyadmin y cambiar el apartado “esAdmin” 
del Usuario al número “1”, indicándole al login en el comento de comparar que es un Administrador. De esta forma ya podremos acceder 
al panel de Administración como un Administrador, luego dentro del panel de Administración, ya se pueden registrar usuarios y elegir
si dichos usuarios serán Administrador o usuarios normales, pero el primer Administrador de la web hay que aplicarlo así.
